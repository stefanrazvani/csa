# Bibliotecă și studiu

Modul multi-tenant pentru lucrări structurate, text direct, versiuni, concepte manuale și dezbateri contextuale.

- MongoDB este sursa canonică.
- `minGrade` este aplicat în metode și publicații, niciodată numai în client.
- publicarea unei versiuni creează un job pentru proiecțiile OpenSearch și ArangoDB;
- OCR nu este implementat; fișierele scanate primesc starea `unsupported_scan` în worker;
- drepturile de stocare și procesare sunt obligatorii înainte de publicare.

Rute: `/biblioteca`, `/biblioteca/:id`, `/dezbatere/:id`, `/concepte` și variantele protejate `/portal/...`.
